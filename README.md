# Team Profile Generator

## Description
A Node app used to generate a webpage that displays employees on a team.
- Sample of the created HTML file is within the /dist folder

**info for each employee**
- title
- name
- email
- employee ID
- role-specific field

## Packages Included
- [Jest]
- [Inquirer]
- [Bootstrap]
